# ClickConvert

![ClickConvert Banner](banner.png)

A fast and simple image converter that works directly from the Windows right-click menu.  
100% free. No ads. No subscriptions. No in-app purchases.

## ✨ Features

- 🖱️ Right-click integration — convert files directly from File Explorer
- 🔄 Supports HEIC, JPG, PNG, WEBP, TIFF, and PDF
- 📦 Batch conversion — process multiple files at once
- 🔌 Fully offline — no uploads, no cloud, no data collection
- ⚡ Lightweight and fast

## 📥 Download

Go to [Releases](../../releases/latest) to download the latest `.exe`.  
No installation required — just run directly.

## 🚀 How to Use

1. Right-click any image file in File Explorer
2. Select **Show more options** → **Convert with ClickConvert**
3. Choose your target format
4. Conversion happens instantly — no app window needed

> **Note:** On Windows 11, third-party context menu items appear under "Show more options".  
> This is a Windows 11 system behavior and applies to all third-party apps.

### Output Location

- **Image → PDF:** Each image is saved as an individual PDF in the same folder
- **PDF → Image:** Output images are saved to `converted/[original filename]_[format]/` in the same folder

## 💻 System Requirements

- Windows 10 / 11
- No additional dependencies needed
